# Reliant Homecare Services Website

This project is a beginner-level website created using HTML, CSS, and JacaScript for the NextChapter Apprenticeship Program.

The website represents a home watch company called Reliant HomeCare Services that me and my wife are starting and demonstrates basic front-end web development skills.

## Project Purpose

The goal of this project is to demonstrate:

- Basic HTML page structure
- CSS styling and layout
- JavaScript interaction with the DOM
- Separating code into multiple files
- Hosting code using GitHub

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- GitHub
